## 1. Describing Motion
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Speed** | $v = \frac{d}{t}$ (Scalar: Distance over time) |
| **Velocity** | $v = \frac{\Delta x}{\Delta t}$ (Vector: Displacement over time; includes direction) |
| **Acceleration** | $a = \frac{\Delta v}{\Delta t}$ (Rate of change of velocity) |
| **Displacement** | The straight-line change in position from start to finish. |
| **Distance** | The total length of the path traveled. |
| **Frame of Reference** | The perspective or coordinate system used to measure motion. |
| **Motion Graphs** | Position-vs-Time (slope = $v$); Velocity-vs-Time (slope = $a$, area = $\Delta x$). |

## 2. Linear Motion
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Newton’s 2nd Law** | $F_{net} = ma$ (Force equals mass times acceleration) |
| **Weight** | $F_g = mg$ (Gravitational force; $g \approx 9.8 \, m/s^2$ on Earth) |
| **Newton’s 1st Law** | Law of Inertia: Objects resist changes in motion. |
| **Newton’s 3rd Law** | Action-Reaction: Forces come in equal and opposite pairs. |
| **Projectile Motion** | Horizontal motion is constant velocity; vertical motion is constant acceleration ($g$). |
| **Free-Body Diagram** | A diagram showing all vector forces acting on an object. |
| **Drag** | Air resistance; a force that opposes motion through a fluid. |

## 3. Circular Motion & Gravitation
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Centripetal Force** | $F_c = \frac{mv^2}{r}$ (Force directed toward the center of the circle) |
| **Universal Gravitation** | $F_g = G\frac{m_1 m_2}{r^2}$ (The force of attraction between any two masses) |
| **Centripetal Accel.** | $a_c = \frac{v^2}{r}$ (Acceleration toward the center) |
| **Inverse Square Law** | If distance doubles, the gravitational force becomes $1/4$ as strong. |
| **Uniform Circular Motion** | Constant speed, but changing velocity (because direction changes). |

## 4. Mechanical Energy and Work
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Work** | $W = Fd \cos\theta$ (Energy added to or removed from a system) |
| **Kinetic Energy** | $KE = \frac{1}{2}mv^2$ (Energy of an object in motion) |
| **Potential Energy** | $GPE = mgh$ (Stored energy based on height/position) |
| **Cons. of Energy** | $E_{total} = KE + PE$; Energy cannot be created or destroyed. |
| **Mechanical Energy** | The sum of kinetic and potential energy in a system. |
| **Power** | $P = \frac{W}{t}$ (The rate at which work is done) |

## 5. Momentum
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Momentum** | $p = mv$ (Vector quantity of motion) |
| **Impulse** | $J = F\Delta t = \Delta p$ (Force applied over time changes momentum) |
| **Cons. of Momentum** | Total $p$ before a collision = Total $p$ after ($m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$). |
| **Elastic Collision** | Momentum is conserved AND Kinetic Energy is conserved. |
| **Inelastic Collision** | Momentum is conserved, but Kinetic Energy is lost (heat/deformation). |

## 6. Electrical Forces & Interactions
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Coulomb’s Law** | $F_e = k\frac{q_1 q_2}{r^2}$ (The force between two electric charges) |
| **Conductor** | Material that allows electrons to flow freely (e.g., copper). |
| **Induction** | Rearranging charges in an object without physical contact. |
| **Electric Charge** | Protons (+) and Electrons (-); like charges repel, opposites attract. |
| **Inverse Square Law** | Electric force weakens rapidly as distance increases. |

## 7. Electric Current
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Ohm’s Law** | $V = IR$ (Voltage = Current $\times$ Resistance) |
| **Electrical Power** | $P = IV$ (Current $\times$ Voltage; measured in Watts) |
| **Series Circuit** | Current is the same everywhere; resistances add up ($R_{total} = R_1 + R_2$). |
| **Parallel Circuit** | Voltage is the same everywhere; current splits between branches. |
| **AC vs. DC** | Alternating Current (wall outlet) vs. Direct Current (battery). |

## 8. Magnetism
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Magnetic Field** | Region where magnetic force is exerted; flows North to South. |
| **Faraday’s Law** | A changing magnetic field induces an electric current in a wire. |
| **Lenz’s Law** | An induced current creates a field that opposes the change that made it. |
| **Generator** | Device that converts mechanical energy into electrical energy. |

## 9. Mechanical Waves
| Topic | Key Concept / Formula |
| :--- | :--- |
| **Wave Speed** | $v = f\lambda$ (Velocity = Frequency $\times$ Wavelength) |
| **Frequency ($f$)** | Cycles per second (Hertz, Hz). $f = 1/T$. |
| **Period ($T$)** | Time for one full cycle. $T = 1/f$. |
| **Transverse Wave** | Medium vibrates perpendicular to wave motion (e.g., light). |
| **Longitudinal Wave** | Medium vibrates parallel to wave motion (e.g., sound). |

## 🧪 Essential Lab & Analysis Skills
*   **Data Analysis:** Be able to calculate the slope of a line to find velocity or acceleration.
*   **Model Evaluation:** Use diagrams to explain concepts like energy transfer in a pendulum.
*   **Experimental Design:** Identify independent, dependent, and controlled variables.
*   **Error Analysis:** Explain why a lab result might differ from the theoretical value (friction, human error, etc.).
